Lino README 
===========

About Lino
----------

Lino is a framework for creating customized enterprise-level 
Rich Internet Applications using Sencha ExtJS and Django 

More on http://lino-framework.org/about/index.html

Community
---------

Lino lives currently from personal enthusiasm of a few people, 
but we hope that the community will grew soon.
This is the time for pionieers to join us!

Lino is published under the GPL because we want to make sure 
that our invested work will always belong to the general public 
and never be misused by some proprietary organisation.

More on http://lino-framework.org/community.html
