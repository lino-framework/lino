Class Ext.ensible.cal.EventEditForm

Package:Ext.ensible.cal
Defined In:EventEditForm.js
Class:EventEditForm
Extends:Object

A custom form used for detailed editing of events.

This is pretty much a standard form that is simply pre-configured for the options needed by the calendar components. It is also configured to automatically bind records of type Ext.ensible.cal.EventRecord to and from the form.

This form also provides custom events specific to the calendar so that other calendar components can be easily notified when an event has been edited via this component.

The default configs are as follows:
